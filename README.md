# ERF Editor

Only the start to a custom ERF Editor


## For Starters

Install dependencies

```
npm install
```

Build react

```
npm start build
```

Start the App

```
npm start
```

Customize the application and build for multiplatform systems

```
npm run build_osx
npm run build_win
npm run build_linux
```
