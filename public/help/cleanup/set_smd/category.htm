Set SMD Attribute

The Set SMD Attribute action is a read/write DFM action that attaches the .smd system attribute to a subset of the pads in a layer. The purpose of this action is to prepare the layer for effective analysis that will be able to separate measurements into different categories according to types of pads. Setting of the attribute is based on the shape of the pads and whether pads are drilled.

Special Reports

Reports the total number of modified pads as well as a histogram by the names of the pads.

Result Attributes

 

Attribute

Description

<layer_name>_set_smd

The number of modified pads in the layer <layer_name>

set_smd

The total number of modified pads in all layers.

For example, lyr1_set_smd , lyr2_set_smd, each provides a total for a specific layer (lyr1, lyr2), while set_smd provides a total for all layers.
