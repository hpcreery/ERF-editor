Legend Detection

Legend Detection (LD), sometimes called Nomenclature Detection, is a read-write DFM action. It is intended to identify and mark features in positive signal and soldermask layers that it identifies as legends.

Legends can be defined as drawn texts which have no electrical significance (such as nets without pads or drills, covered by soldermask, etc.). Legends detected as such by this action are tagged with the .nomenclature attribute. Features that are tagged manually by the operator with the .nomenclature attribute will also be considered as legends by this action and excluded from analysis actions together with all detected legends.

Special Reports

Totals for `Nomenclature Features' category for each layer are reported.

Result Attributes

Result attributes according to the following syntax are produced:

Result Attribute

Description

<layer name>_nomenclature

Number of nomenclature elements in layer
