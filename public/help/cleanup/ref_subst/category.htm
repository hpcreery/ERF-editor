Construct Pads (Ref.)

The Construct Pads (Ref.) action converts drawn pads to flashed pads based on one or more groups of features selected manually by the operator prior to activating the action.

Unlike its sister action, Construct Pads (Auto.), no attempt is made to heuristically identify the groups which will form the reference for construction. It solely depends on the operator to make this distinction. As a result, the operation tends to be much faster and more accurate. The obvious drawback is the need for operator intervention.

Special Reports

The names and counts of each pad created is listed in the report.

Result Attributes

 

Attribute

Description

<layer_name>_construct

The number of constructed pads in the layer <layer_name>.

construct

The total number of constructed pads in all layers.

For example, lyr1_construct , lyr2_construct each provides a total for its specific layer ( lyr1, lyr2 ), while _ construct provides the total for all layers.
