Construct Pads (Auto, All Angles)

This function identifies drawn pads (including ones drawn at non-90 degree rotation),
and translates them into flashed pads. You may control what types of pads may be created.

Note: the ability to build Pads in all angles was added to the Construct Pad (auto) action,
and you can use that action instead, if desired..

If most of the drawn pads are placed horizontally or vertically use
    DFM -> Cleanup -> Construct Pads (Auto.)

If the features are drawn with pads instead of lines run
    Edit -> Reshape -> Pad To Line on these features first. 

To access the Construct Pads Action:
• Activate the Graphic Editor
• Select DFM > Cleanup > Construct Pads (Auto., All Angles)

  
This function runs on one or more layers at once and has the following
parameters which control its behavior:

Layer:        layer name or layer expression

Apply to:     work on all features or on selected features only

Minimum:      smallest drawn feature to consider

Maximum:      largest drawn feature to consider

Tolerance:    amount of variance in edges to allow 

Reference SM: solder mask layer name

Standard Symbol Allowed : types of standard symbol that allowed for creation

The reference solder mask is optional. It can help the pad recognition
and reduce the number of false alarms by considering the pads on the
solder mask layer.
Note: construct pads should be run on the solder mask layer first.
