Profile Checks

The Profile Checks action is a read-only action that  measures distances between features and inner profiles (step & repeat PCBs), and distances between features and outer profiles (PCB and panel), and generate statistics.

The check can operate on any layer.

Tests

The following tests can be applied to profile layers, as listed in the action screen and below:

  • Size - categories for this test:

         - Board length

         - Board width

         - Board thickness

  • Spacing -

         - Feature to profile

         - Feature to Step & Repeat

  • Gaps - Conveyed edge gap

Attributes

There are no Feature attributes used during Profile Checks.

Result Attributes

 Attribute

Description

min_f2outer

Minimum distance between PCB features and PCB profile.

min_f2inner

Minimum distance between panel features and profile of encompassed PCB.
