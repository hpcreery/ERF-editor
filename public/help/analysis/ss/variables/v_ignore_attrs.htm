ERF Variable

Description

v_ignore_attrs

This variable enables you to specify a list of attributes. Features having at least one of these attributes are ignored by the action.
The attributes can be system attributes or user-defined attributes.
Multiple attributes should be separated by a semicolon (;) or a comma. The length of this ERF variable is limited to 256 characters.

Note: The system attribute ignore_action can be assigned to individual features. Any feature possessing this attribute is ignored by the action. This attribute is useful if a specific feature has none of the other attributes defined in the ERF variable v_ignore_attrs.

The ignore_action attribute must be specified in the list of attributes defined in the ERF variable v_ignore_attrs in order to enable it.
