max_line_width

Maximum line width of lines in line category

min_line_len

Minimum line length of lines in line category, also minimum overlap between lines to report in parallel spacing

len2width_ratio

Minimum ratio of len to width of a line to be inserted into the line category

v_report_include_
sm_as_zero

When 0, the distance between the soldermask clearance that is included inside a silkscreen element is reported as its real distance in the SM Clearance category.

When 1, the distance segment between the soldermask clearance that is included inside a silkscreen element becomes a zero length distance segment, and is therefore reported in the red severity level of the SM Clearance category.

v_report_arcs_
with_lines

This variable controls in what category arcs and shaved arcs are reported.

No = arcs are reported in the new ss_arc category. Shaved arcs are reported in the new ss_shaved_arc category.

Yes = arcs are reported in the ss_line category with lines reports. Shaved arcs are reported in the ss_shaved line category along with shaved lines reports.

Note: The Arcs and Shaved Arcs reports are subject to line reports limitations: max_line_width, min_line_len, len2width_ratio.

v_check_only_
exposed_drill

This variable enables you to define what drill holes to check.

Yes (the default value) = the action checks only exposed drill holes.

No = all drill holes are checked.

v_report_include_
as_zero

This variable enables you to define how to report Silk Screen elements which are fully included inside drill or rout features.

No (the default value) = the action reports the measured distance.

Yes = the distance will be reported as zero length distance that will touch the SS element.
