Purpose

To allow the specification of different pad-to-pad spacing requirements to each component type.

Function

Reports pad-to-pad spacing measurements less than the value specified in the component attribute .toep_spacing_req (within the tolerance specified in the ERF variable toe2toe_tol ). This applies when the ERF variable classify_pad2pad = Yes, and the component that is on top of both pads being measured has the attribute .toep_spacing_req.

Activation

When ERF variable classify_pad2pad = Yes
