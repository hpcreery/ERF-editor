Function

This report gives the user an additional tool that could point to possible problematic locations if the Drill hole shifts.

The distance is between the entrance point of the line's segment to the Pad and Drill hole which pierces that Pad.

Note: The entering trace can only be a line feature. The pad cannot be a drawn pad

Activation

The reports in this category are controlled by the ERF variable v_it2h_report_mode .

When v_it2h_report_mode = 0, no reports are made in this category (The default value of this variable).

When v_it2h_report_mode = 1, all cases are reported.

When v_it2h_report_mode = 2, reports only cases where the distance measured is equal or less than the AR

measured to this Pad + reg_allowance.
