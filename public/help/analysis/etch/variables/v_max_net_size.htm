ERF Variable

Description

v_max_net_size

Determines whether or not to create "etched layers", and in what situations "etched layers" should be created. The options are:

No - do not add etched layers. (Default)

Reported - only add an etched layer for those layers where problems are reported.

All - add an etched layer for each layer checked. This can be useful when trying to understand why an expected report is missing, or for finding 'Unprocessed' nets because all shrunken features display the shrink outline. Those features that could not be processed have no outline.

Note: When set to Reported or All, during results display the original layers are displayed with their the corresponding etched layers.
