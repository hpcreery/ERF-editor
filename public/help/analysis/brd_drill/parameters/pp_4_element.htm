Second Element Type (PTH)

pp_2_element

Define against which holes you want to check vias

 

Possible values are NPTH , Via , PTH, Rout and .all or blank (If left blank than also pp_1_element should be left blank.

 

Rout is defined to be any element on the Rout-layer.

Rout-element attribute overrides other attributes: i.e. the NPTH element on the Rout layer is treated as a Rout-element

.all specifies to select all possibilities. A minus sign defines to deselect the specific type (only in combination with .all)

 

Examples:

pp_2_element = .all      (run all tests)

 

pp_2_element = .all;-Rout;-NPTH (run all checks except against NPTH and Rout)

 

pp_2_element = PTH;NPTH (only check against PTH and NPTH)
