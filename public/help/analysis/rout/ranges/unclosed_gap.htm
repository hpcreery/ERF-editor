Rout Analysis Action

Unclosed Gap reports any two adjacent features in a rout chain whose skeletons are not connected. 

Item

Details

Full Name

Unclosed Gap

Internal ERF Name

unclosed_gap

Measurement Shape

Segment between two skeletons of rout features in the chain

Measurement ID(s)

None

Layers Displayed

Rout Layer

 

Purpose

Avoid jump of router between two nearby rout features

Function

Report about two nearby features in rout chain that are not connected in their skeleton. Any spacing between two nearby features that is bigger then v_min_unclosed_gap and is smaller than v_max_unclosed_gap will be reported in this category.
