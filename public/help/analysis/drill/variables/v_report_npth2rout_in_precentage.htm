ERF Variable

Description

v_report_npth2rout_in_precentage

 Control how the report of the range NPTH2Rout is sent.
NPTH2Rout  reports all non-plated thru holes which are close to the rout path.
If NPTH2Rout  = yes,  the distance is reported in percentage.
If NPTH2Rout  = no, the distance is reported as real distance.
