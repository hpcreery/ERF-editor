Orbotech AOI Checks

This chapter describes the checks made when performing AOI actions on Orbotech machines.

 

Screen Parameters

Max Clearance AR

Maximum clearance permitted between adjacent annular rings. 

Max NFP Spacing

Maximum spacing permitted between adjacent non-functional pads.
 

Test List

The following tests can be applied to affected layers. Measurements used for these tests are defined by the parameters defined above.

Clearance Detection - detects clearance between adjacent annular rings.

NFP Spacing - Detects spacing between adjacent non-functional pads.
