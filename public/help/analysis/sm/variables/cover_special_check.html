ERF Variable

Description

cover_special_check

0 = do not check.

1 = Check for two intersections in SM clearance where the first is legal and the second connection to copper causes a violation.

Note : cover_special_check requires considerable processing time.

2 = Adds new tests to existing coverage tests.

*    Consider partially covered pads (i.e. pads whose center is covered by the soldermask clearance) as fully covered, and do not report the (partially covered) pads or the traces coming from them as coverage problems.

*    Check for a coverage problem on traces (if they exist) that leave a soldermask clearance and other soldermask clearances (related to it) that are in the same soldermask net.

*         The new checks may increase analysis run time.

3 = Checks for surfaces that intersect SM clearances.  

      Checks for coverage problems between the edges of the surface and the SM clearance.

 

If the ERF variable v_report_min_coverage = yes, the action reports only one coverage - the minimum coverage distance.
If v_report_min_coverage = no, the action reports all coverage problems between surface edges and the SM clearance.
