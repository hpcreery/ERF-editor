Drill Summary

Drill Summary is a read-only action intended to generate statistics on holes in padstacks. It finds the minimum annular ring in a padstack and generates a table listing all the annular rings in the padstack for a specific hole.

Tests

Measurements

Measurements are from edge of hole to other net copper encompassing copper or SM clearance.

Dynamic Categories

The Drill Summary action uses dynamic category names to report annular rings and hole dimensions.

The general structure of a category name is as follows:

layer_name

outer
inner
smask

PTH
VIA
