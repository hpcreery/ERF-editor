Function

Reports for PTHs embedded in copper the shortest distance between PTH and laminate, if the distance is less than the pp_d2c action parameter. Takes into account both negative and positive data. Measurements are spacing segments representing the shortest distance from the drill to laminate.

Activation

Select the Drill test and check that the classify_pad_ar ERF variable is set to Yes.

Known Limitations

Since, in drawn surfaces, no single pad gives the PTH an annular ring value, then an r0 ID will be inserted.
