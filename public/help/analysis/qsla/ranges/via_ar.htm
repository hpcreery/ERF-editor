Function

Reports for vias embedded in copper the shortest distance between via and laminate, if the distance is less than the pp_d2c action parameter. Takes into account both negative and positive data. Measurements are spacing segments representing the shortest distance from via to laminate

Activation

Select the Drill test and ensure the classify_pad_ar ERF variable is set to Yes .

Known Limitations

If no single pad gives the via the same annular ring value as that calculated for all copper elements, then no pad will be associated with the measurement and an r0 will be inserted instead.
