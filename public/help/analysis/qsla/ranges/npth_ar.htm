Function

Reports for NPTHs embedded in copper the distance to the nearest laminate that is less than the pp_d2c action parameter.

If limited_npth_check is set toNo , all annular rings in the vicinity are reported. If set to Yes , the maximum distance reported is that defined by pp_d2c .

Measurements are distance segments between the NPTH and nearest laminate.

Activation

 

Known Limitations

The action tries to identify the pad that is associated with the NPTH and add its ID to the measurement. If it cannot (e.g. NPTH drilled thru drawn features), it inserts an r0 instead.

If there is any copper in the NPTH, then a zero annular ring measurement will be reported. This may be considered by some to be a false alarm.
