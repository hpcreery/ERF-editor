Function

Reports width of lines representing conductor traces in signal layers. Lines with the attribute `.pattern_fill' and '.nomenclature' are not reported. If a line is drawn (represented by two touching lines running along each other) the compound width of the drawn line is reported.

Lines will be reported that are:

·    Longer than the min_line_len ERF - variable

·    Narrower than the max_line_width ERF - variable

·    Their Length to Width ratio is greater than the len2width_ratio ERF-variable.

Activation

Select the Size test.
