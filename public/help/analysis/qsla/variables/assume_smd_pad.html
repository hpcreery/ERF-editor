ERF Name

 Description

assume_smd_pad

Define if the Quote Analysis action should assume that non-drilled pads and  SMD pads will be used in the board for which a quote is being created.
When assume_smd_pad = No (Default), the Quote Analysis action should assume that non-drilled pads and  SMD pads are not used.
When assume_smd_pad = Yes, the Quote Analysis action should assume that non-drilled pads and  SMD pads  are used.
