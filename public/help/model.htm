Parameter

Description
Model Name

The Model name to be assigned to this model

Units

Units in which the values will be written to the file when it is being saved.

Comments

Enter free text to define the model

Range colors

Up to 5 ranges are allowed. In order to activate a range, select the checkbutton before the range color. The press the “Modify…” button to select the appropriate color for this range.

Modify…

Select Modify to change the color to be assigned to this range
