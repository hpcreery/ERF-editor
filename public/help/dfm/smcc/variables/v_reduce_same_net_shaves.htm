ERF Variable

Description

v_reduce_same_net_shaves

Yes = the action activates an additional check that avoids unnecessary same net shaves.
No (default) = the action does not activate the additional check.

 

Shaves against lines that are part of a drawn area and have no air gap between them and the corresponding signal pad.

Shaves against lines touching lines coming out of the Pad. If those lines create a coverage violation with the clearance, then a shave will be placed to solve this coverage problem. This additional check will distinguish which lines need this coverage shave.

Only lines that have a 90-degree angle with the distance segment from the clearance will be shaved.

 

Notes:       

This additional check checks only unshaved lines.
This additional check can increase action run time.
