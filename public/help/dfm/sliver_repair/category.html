Sliver Repair

Introduction

Sliver Repair is a DFM action designed to handle sliver problems. It is a combination of all the current DFM actions that handle different kinds of sliver issues. This action contains several enhancements to the older algorithms. It can also run in Report mode, in which it only reports the sliver problem, but does not repair it.

This new DFM action requires the dfmsfil license option, the same license that is required today for all Sliver DFM actions.

The action is part of Sliver actions sub-menu in the Graphic Editor:
DFM-> Sliver -> Sliver Repair .

Glossary

Sliver

A spacing between positive features which can be filled by adding a positive feature.

Sliver Width

the spacing value between the layer features.

Sliver Length

the length of the patch needed for filling a sliver.

Local Spacing

the area in the sliver which cannot be filled.
