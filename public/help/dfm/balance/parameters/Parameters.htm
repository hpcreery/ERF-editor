Parameters

Screen Name

Parameter

Type

Description

Layer

pp_layer

layer

Defines which layers should be processed. Its value may be a single layer name or an expression. Any layer may be used.

Shape

pp_shape_type

radio

Values = Circle, Square, Diamond, Symbol.

Defines shape of feature to be used in balancing. Three standard types of shapes are supported:

*    *    Circle - represented by zero-length line drawn with round aperture.

*    *    Square - represented by zero-length line drawn with a square aperture.

*    *    Diamond - represented by a diamond shaped aperture3

*    *    Symbol - any symbol, standard or special, can be used to produce thieving features.

Shape Size

pp_size

p_mil

Range = 10 <-> 500

Defines shape size for standard thieving shapes (see description of the Shape parameter below). The following factors are used:

Circle - circle diameter

Square - side length

Diamond - height.

Restriction:

The Shape Size value is ignored if Symbol is selected as the value for the Shape parameter.

Pitch

pp_pitch

p_mil

Range = 10 <-> 500

Defines the distance between standard thieving shapes (see description of the Shape parameter). In standard shapes, symmetric pitch applies to both the X and the Y dimensions.

Restriction: Pitch value is ignored if Symbol is selected as the value for the Shape parameter.

Drill Spacing

pp_drill_spacing

p_mil

Range = 10 - 500

Defines distance to holes drilled through the layer

Feature Spacing

pp_feature_spacing

p_mil

Range = 10 - 500

Minimal distance to maintain from any layer feature

Rout Spacing

Profile Spacing

pp_prof_spacing

p_mil

Range = 0 - 500

Minimal distance to maintain from the profile

Fiducial Spacing

pp_fiducial_spacing

p_mil

Range = 0 - 500

Minimal distance to maintain from fiducials (features assigned the .pad_usage fiducial attribute).

Mask Layer

pp_mask_layer

layer

Any layer can be used to define keep away areas. If this parameter field is not blank, its value is supposed to be the layer name. No thieving feature will be added if it touches any feature in this layer.

Symbol Name

pp_shape_name

string

If Symbol is selected as the value of the Shape parameter, the value of this field will be considered a name of a standard or a special symbol.

A surface with the shape of this symbol will be used as the thieving feature.

Restriction: If no standard or special symbol with the specified name exists, an error exit is performed.

Symbol Pitch X

pp_pitch_x

p_mil

Range = 10 - 500

If Symbol is selected as the value of Shape Pitch X, define pitch in the X dimension.
