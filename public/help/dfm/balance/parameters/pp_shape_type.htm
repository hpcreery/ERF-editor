Shape

pp_shape_type

radio

Values = Circle, Square, Diamond, Symbol, Solid.

Defines shape of feature to be used in balancing. Four standard types of shapes are supported:

     Circle - represented by zero-length line drawn with round aperture.

    Square - represented by zero-length line drawn with a square aperture.

    Diamond - represented by a diamond shaped aperture3

    Symbol - any symbol, standard or special, can be used to produce  features.

  Solid - enables adding solid copper surfaces.
