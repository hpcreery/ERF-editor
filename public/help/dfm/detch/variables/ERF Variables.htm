ERF Variables

ERF Name

Description

symbol_rounding_value

Symbol of a compensated feature will be rounded with this value.

smd_min_ratio

The minimal ratio allowed between the dimensions of the SMD's long side and short side. If this value is set to 0, the test is disabled.

min_edge_length

This replaces the ERF variable min_seg_length when used for the edges of decomposed surfaces and flattened pads.

min_seg_length

A line may be broken into segments, taking into consideration other neighboring features. Such segments will be at least min_seg_length long.

compensate_imp_lines

Yes: compensate a line that has a .imp_line attribute in the same manner as a regular line.

No: don't change a line that has a .imp_line attribute.

consider_same_net_
spacing

Yes: Features on the same net will affect the compensation value as will features from different nets.

No: Features on the same net will not affect the compensation value.
