Legacy contains DFM actions that have been used for a long time, but have been superseded by more advanced actions. These classic DFM actions are maintained for use by those customers who need them.

Classic Sliver Fill

Classic Sliver Fill is a read-write DFM action that fills slivers between positive features, by adding square-capped lines to fill the slivers it detects. Only slivers created by shapes from the same net are filled, to avoid introducing electrical shorts into the circuit. Sliver Fill operates on any layer. In negative layers (with the exception of power & ground layers) polarity is reversed.

Limitations

When there are more than a certain number (25) of neighbors for a single shape, no action is taken for this shape. Also, slivers with acute angles are not treated.

Note: In certain cases, this function may leave tiny holes. If these holes are formed, run the Pinholes Elimination DFM action to remove them.
