Ground Pad Diam

pp_gnd_size1 pp_gnd_size2 pp_gnd_size3

p_mil

Range: 0 <-> 200

These are the diameters that define ground pads when mode Diameter is specified in the parameter
Ground Pads recognition by.

To use only one or two of the fields, set all other fields of the parameters to 0 (zero).

Either way, the Ground pads must be round and must not have the .smd or .test_point attributes.
