Parameters

 

Screen Name

Parameter

Type

Description

Layer

pp_layer

layer

Defines the layers to be processed. Its value may be a single layer name or an expression. The paste layer or the signal/ mixed layers that are defined as top or bottom layers will be used.

Width Mode

pp_w_mode

menu

Distance, Factor, Value
Defines how SMD pads will be shrunk: by width, distance, factor or value.

Factor (Width Mode)

pp_w_fact

double

Range: 0.3 <-> 0.99

When the Width Mode parameter is set to Factor, this field contains the factor by which the SMD pad's width will be divided.

Value (Width Mode)

pp_w_val

p_mil

Range: 2 <-> 500
When the Width Mode parameter is set to Distance, this parameter is the distance that the width of the SMD pad will be shrunk. When the Width Mode parameter is set to Value, this parameter is the absolute value by which the SMD pad width will be changed.

Affect selected

pp_sel_mode

radio

Range: No, Yes

Yes - specifies that the SPO DFM action will apply only to user-selected features.

No - applies to all features in the layer. The features must be selected from the Work layer.

Ground Pads recognition by

pp_gnd_recognize

radio

Range: None, Diameter, Selection

Defines how to identify Ground pads.

None - completely ignored.

Diameter - identified by diameter (for pads that do not have the .smd or .test_point attributes).

Selection - selected pads (without .smd or .test_point attributes), and the Affect Selected parameter (above) is forced to No.

Ground Pad Diam

pp_gnd_size1, pp_gnd_size2, pp_gnd_size3

p_mil

Range: 0 <-> 200

These are the diameters defining Ground pads when Diameter mode is specified in Ground Pads Recognition By.

To use only one or two of the fields, set all other fields of the parameters to 0 (zero). Either way, the Ground pads must be round and must not have .smd or .test_point attributes.

Ground Pad AR

Gap
(Paste Width)

pp_gnd_gap
