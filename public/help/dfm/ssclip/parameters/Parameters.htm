Screen Name

Parameter

Type

Description

Screen Layers

pp_silk_layers

layer

Defines which layers will be affected by the action, according to the syntax defined in Doc.0501, Checklist Operations. Only silk screen layers will be considered. For each silk screen layer relevant soldermask drill and signal layer are processed.

Mode

pp_clip_or_merge

radio

Values= 'Clip', 'Merge' and 'Clip & Merge'

One of the three processing modes to be used.

Clip Silk Screen Against

pp_option_set

set

Options = Solder Mask,

Drill Pth,Drill Npth,

Drill Via, SMD Pads, Pads

Defines which features will influence silk screen optimization process. Any combination of options may be selected.

SMD Pads and Pads = These options enable the action to use either pads or SMD pads in the corresponding outer layer for clipping or merging. In order to use these options be sure that the ERF variable v_process_outer_layer = Yes .

Min Solder Mask Dist

pp_min_sm_distance

p_mil

Range= 0 - 1000.

Defines threshold distance between soldermask clearance edge and clipped/merged silk screen feature.

Min Pth Drill Dist

pp_min_drl_distance_pth

p_mil

Range= 0 - 1000.

Defines threshold distance between plated holes and clipped/merged silk screen feature.

Min Npth Drill Dist

pp_min_drl_distance_npth

p_mil

Range= 0 - 1000.

Defines threshold distance between non-plated holes and clipped/merged silk screen feature.

Min Via Drill Dist

pp_min_drl_distance_via

p_mil

Range= 0 - 1000

Defines threshold and clipped/merged silk screen feature.

Min Outer Layer Dist

pp_min_signal_distance

 

This parameter defines the threshold distance between the pads edge and the clipped/merged silk screen features.

Min Line Width

pp_min_line_width

 

This parameter enables you to define a minimum line width. Any line that has a line width less than the minimum will be enlarged to the minimum. The enlarged lines are reported in the category Enlarged_Lines.

Use Reference Features

pp_use_selected

 

This parameter enables you to select reference features for the merge or clip.
